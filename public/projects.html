Password Manager

Github

Password Manager is an application that can generate, encrypt, and securely store credentials for accounts.

  • Written in C++ with OpenSSL and wxWidgets.
  • Features a customisable and cryptographically secure password generator.
  • Encrypts all data with AES-256 (CBC mode) and leaves no trace of plaintext data in memory after execution.
  • Provides an efficient search function and user-friendly UI, making it simple to access account information.
Crazy Engine

Github

Crazy Engine is a 2D game engine capable of rendering both graphics and sound.

  • Written in C++ with OpenGL, Glad, and the Win32 API.
  • Provides an efficient batch renderer with functions for rendering colours, textures, and strings with GLSL shaders (default or user-provided).
  • Uses the Win32 WASAPI audio framework to render sound on Windows.
  • Features a custom-built mathematics library.
Math Script

Github

Math Script is a tool for generating properly-typesetted mathematical formulae from an input script.

  • Written in C++ with Freetype.
  • Uses a custom scripting language to generate bitmaps of inputted formulae.
  • Supports inline operations, fractions, integrals, summations, text, and any ASCII symbol.
  • Can evaluate multivariate functions involving arithmetical, fractional, and trigonometric operators with given input parameters.
Crazy Engine GUI

Github

Crazy Engine GUI is an extension of Crazy Engine that adds support for GUI development.

  • Written in C++ on top of Crazy Engine.
  • Provides an array of UI primitives, including buttons, checkboxes, images, and labels.
  • Supports a scaling GUI with relative translations and sizes.
